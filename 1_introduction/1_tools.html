Evolutionary Computation

Tools and format of the class


Course overview


Two main components:
  • a tutorial overview of evolutionary computation
  • a project focused on game playing with evolved agents


Tutorials:
  1. Introduction
  2. Evolutionary Strategies
  3. Multi-Objective Evolution
  4. Evolution of programs, graphs, and networks
  5. Genetic representation and operator design
  6. Evolution and behavior
  7. Evolution with constraints

Course format


  • Follow tutorials individually or in groups
  • Move on to next section on completing exercises
  • Evaluation by pair: you report on a classmates's notebook
  • I'll be available during class hours on Discord
  • Different format for guest lectures

Course tools overview


  • Discord for chat and asking questions
  • Zoom for meeting
  • Git for code sharing and class organization
  • Julia for tutorial code
  • Jupyter for tutorials


This class is oriented towards a Linux environment, but everything should work on Windows. If you're new to Linux, you can use a virtual machine inside of Windows or OS X; Ubuntu is recommended.

Git


  • A popular Version Control System (VCS)
  • Publicly available git servers: github, gitlab, and others
  • Get set up on github
  • Run directly from a shell or use a GUI


Julia


  • Scientific computing language; fast but easy to read and understand
  • Extensive documentation, classes, and active community
  • Interactive development workflow with Juno IDE, Jupyter, REPL

Jupyter


  • Ju (julia) pyt (python) er (R)
  • Notebooks: code and text in the browser
  • Local server with the kernel type
  • IJulia.jl

  • Create a notebook in your fork that prints "Hello world"
